# Fifty Challenging Problems in Probability

My solutions to the problems presented in the classic book *"Fifty Challenging Problems in Probability with Solutions"* by Frederick Mosteller.

## Overview
This repository contains Jupyter notebooks with Python implementations and detailed explanations for the probability puzzles. Each notebook corresponds to a specific problem or set of problems.

## Tools Used
- Python
- Jupyter Notebooks
- Matplotlib/Seaborn (for simulations and visualizations)
